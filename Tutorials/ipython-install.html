Installation

Use Anaconda from continuum.io - http://continuum.io/downloads . Use the installer for windows, mac or linux. Be sure to use 2.7 python (Do not use 3.4 ) After the installation, find the packages installed by thing the command line instruction "conda list". Look at the available packages from continuum.io as well as the defaults: http://docs.continuum.io/anaconda/pkg-docs

Python test

Next try out the ipython notebook interface. Details of ipython can be found at: http://ipython.org. My current versions is 3.2 but other versions will be documented at this site. At the command line run: 
mkdir notebooktutorial
cd notebook tutorial
ipython notebook
This should open a browser window at localhost:8888/tree. Create a new python 2 notebook and under file rename it Test1. in the IN [1]: box enter: 
In [1]: a=2
        a
The result should show as: 
Out [1]: 2
Using spyder ( a python IDE that comes with anaconda ) Create a new python file with: 
a=2
print(2)
Open a python or ipython console and press the run button. The output should show 2 after the runfile('path/nameofile.py', wdir='path/of/working/directory') is automatically invoked.
